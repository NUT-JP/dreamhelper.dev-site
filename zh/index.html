自动运行（Live）

支持 Free Live / Multi Live / Challenge Live，可配置曲库并辅助周回。

  • 筛选未 Full Combo 的曲目收集星星（Free Live）
  • 限定从收藏曲目中选曲
  • Live Boost 不足时自动恢复（星星/饮料）

手动指定（Manual Play）

手动选择歌曲与难度并运行（例如 Stage Challenge）。

  • 搜索并选择曲目
  • 设置难度并运行

实用工具

自动化任务（如：购买歌曲、角色训练等）

已知限制

模拟器卡顿、微冻结或噪声可能导致时序偏移，使自动演奏不稳定。 低配环境下成功率可能下降。

Multi Live 中，同名曲目（如：オレンジ、シルヴ・プレジデント）可能导致选曲失败， 约 50% 概率失败。 此外，Multi Live 难度目前固定为 EXPERT（如有需求可扩展）。

环境与推荐设置

必需

  • Windows
  • 带 ADB 的 Android 模拟器
  • MuMu / LDPlayer（已测试）

游戏内推荐

  • 速度：9.30
  • 关闭 3D cut-in
  • 动作模式：轻量
  • 开启 FAST/SLOW 与 Perfect 显示
  • 分辨率：1280×720

FAQ

FAQ

为什么经常失败？ +
模拟器卡顿、微小冻结或噪声会导致时序偏移。请检查分辨率与游戏设置，并根据需要调节延迟参数（如 Photogate）。
Multi Live 同名曲目导致失败 +
由于设计限制，同名曲目可能无法正确选择， 约 50% 概率失败。 （如：オレンジ、シルヴ・プレジデント）
