Windows • ADB + MaaFramework • OCR + 模板匹配

在模拟器上辅助 Garupa 操作的自动化工具

Dream Helper（Autodori）通过 ADB 控制 Android 模拟器， 结合 OCR 与模板匹配识别界面状态，并生成触控输入来辅助选曲与运行 Live。

概览

  • 自动连接模拟器并运行流程
  • OCR 识别 + 歌曲目录辅助选曲
  • 生成触控指令进行自动演奏（依赖环境）
  • 支持 Free / Multi / Challenge Live

已知限制

模拟器卡顿/界面差异可能导致识别与时序偏移。 Multi Live 在歌曲同名场景可能出现选曲失败。

环境与推荐设置

必需

  • 带 ADB 的 Android 模拟器
  • MuMu / LDPlayer（已测试）

游戏内推荐

  • 速度：8.0
  • 关闭 3D cut-in
  • 轻量模式
  • 开启 FAST/SLOW 与 Perfect 显示
