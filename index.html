ガルパの操作をエミュレータ上で支援する、 自動化ヘルパー

Dream Helper（Autodori）は、Windows 上で Android エミュレータを操作し、 楽曲選択からライブ実行までの一連の操作を支援する自動化ツールです。

入手方法を見る 注意事項

重要： 本ツールは非公式であり、ゲーム運営・権利者とは関係ありません。 利用は自己責任で行ってください。

できること

  • エミュレータへ自動接続してライブを実行
  • 自動プレイでスターを大量収集
  • ライブブースト自動回復機能でイベントランキング上位へ
  • フリーライブ / マルチライブ / チャレンジライブに対応
  • キャラクターの自動訓練、エピソード自動回収機能搭載(より多くのスターを獲得)

主な機能

UI から Start/Stop と設定を操作できます。

自動プレイ（ライブ）

フリーライブ / マルチライブ / チャレンジライブに対応。曲の候補を絞り込み、周回を支援します。

  • 未フルコンボの曲だけに絞ってスター回収（フリーライブ）
  • お気に入り曲から自由に選曲
  • ライブブーストが足りない場合の自動回復（スター/プチブースト）

Manual Play（手動指定）

曲名と難易度を指定して実行できます。ステージチャレンジ等の用途を想定。

  • 検索して曲を選択
  • 難易度を設定して実行

ユーティリティ機能

自動化タスク（例：楽曲購入、育成タスク等）

既知の制限

端末スペックや小さなフリーズ、ノイズなどでタイミングがずれると自動演奏が不安定になります。 特に低スペック環境では成功率が下がる可能性があります。

マルチライブでは、同名曲（例：オレンジ / シルヴ・プレジデント）が存在する場合、 仕様上 1/2 の確率で失敗するケースがあります。 また、マルチライブの難易度は現状 EXPERT 固定です（需要があれば拡張予定）。

動作要件・推奨設定

必須

  • Windows（本ツールは Windows 前提）
  • ADB が有効な Android エミュレータ
  • MuMu または LDPlayer（対応実績）

推奨（ゲーム内）

  • スクロール速度：9.30
  • 3Dカットイン：OFF
  • 動作モード：軽量
  • FAST/SLOW表示、Perfect表示：ON
  • 解像度：1280×720

よくある質問

FAQ

うまく自動演奏できない／失敗が増えた +
エミュレータのフリーズや処理落ち、背景のノイズ等で入力タイミングがずれると成功率が下がります。 解像度（16:9）やゲーム内設定（軽量モード等）を見直し、必要に応じて遅延調整（Photogate など）を行ってください。
マルチライブで同名曲があると失敗することがある +
仕様上、同名曲が複数あるケースでは正しく選曲できず、約 1/2 の確率で失敗する場合があります。 （例：オレンジ、シルヴ・プレジデント など）
安全ですか？BAN されますか？ +
本ツールはゲームの自動操作を行うため、利用規約に抵触する可能性があります。 いかなるリスクも保証できません。自己責任で利用してください。
