Features

主な機能

全自動ライブ実行

選曲・演奏・結果処理まで全て自動。対応モードを選んで Start するだけ。

リソース自動回復

ライブブースト/エネルギーが不足時に自動で回復。長時間の放置周回が可能。

スケジュール＆放置

就寝前に Start、起きたらイベントポイントが貯まっている。完全放置運用。

安全設計

アプリ改造ゼロ。ADB経由のタッチ入力のみ。ゲームデータに一切触れない。

手動指定モード

曲名と難易度を指定して実行。ステージチャレンジ等の用途にも対応。

ユーティリティ

楽曲購入、キャラ育成、エピソード回収など自動化タスクも搭載。

Requirements

動作要件

必須環境

  • Windows（本ツールは Windows 前提）
  • ADB が有効な Android エミュレータ
  • MuMu / LDPlayer（対応実績あり）

推奨設定（ゲーム内）

  • スクロール速度：9.30
  • 3D カットイン：OFF
  • 動作モード：軽量
  • FAST/SLOW・Perfect 表示：ON
  • 解像度：1280×720

FAQ

よくある質問

BANされない？安全ですか？ +
アプリの改造（判定改ざん等）は一切行いません。ADB 経由でタッチ入力を送るだけなので、ゲームのファイルやメモリには触れません。 2025年10月のリリース以降、BAN報告はゼロです。ただし自動操作である以上、利用規約に抵触する可能性はあります。自己責任でご利用ください。
うまく自動演奏できない / 失敗が増えた +
エミュレータのフリーズや処理落ちで入力タイミングがずれると成功率が下がります。 解像度（16:9）やゲーム内設定（軽量モード等）を見直し、必要に応じて遅延調整を行ってください。
プロセカはどこまで対応していますか？ +
現在はイベント周回・ランクマッチの自動化に対応しています。ガルパほどの機能はまだありませんが、順次拡張中です。Discord で最新状況をご確認ください。
マルチライブで同名曲があると失敗する +
仕様上、同名曲が複数あるケースでは正しく選曲できず、約 1/2 の確率で失敗する場合があります（例：オレンジ、シルヴ・プレジデント）。
