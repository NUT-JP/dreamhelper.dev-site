FAQ

Frequently Asked Questions

Will I get banned? Is it safe?+
Zero app modification — no judgment tampering, no memory editing. Only ADB touch inputs are sent, so game files are never touched. Zero bans reported since the October 2025 launch. However, automation may still violate terms of service. Use at your own risk.
Auto-play keeps failing+
Emulator lag and micro-freezes can break timing. Check your resolution (16:9) and in-game settings (lightweight mode), and tune latency if available.
How far along is Project SEKAI support?+
Event grinding and ranked match automation are live. Feature set is still expanding. Check Discord for the latest status.
Multi Live fails with duplicate song titles+
By design, duplicate song titles may cause selection failures with roughly 50% chance (e.g., Orange, Silhouette President).
